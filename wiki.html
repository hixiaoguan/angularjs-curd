获取文章列表 获取文章总数 添加文章 删除文章 获得一篇文章 更新文章 返回码说明
获取文章列表

调用示例

{
    "action":"get_list",
    "where":"typeid = 1",
    "offset":0,
    "rows":10,
    "orderField":"id",
    "orderBy":"DESC",
}

参数说明

参数 是否必须 说明
action 是 操作方法 get_list
where 否 查询条件的字符串
offset 否 从全部记录的该偏移位置开始返回，0表示从第一条记录返回
rows 否 返回记录的数量，取值在1到100之间
orderField 否 排序方式的字段
orderBy 否 排序方式： DESC(默认) ASC

返回说明

返回符合查询条件的记录
获取文章总数

调用示例

{
    "action":"get_total",
    "where":"typeid=2"
}

参数说明

参数 是否必须 说明
action 是 操作方法 get_total
where 否 查询条件的字符串

返回说明

返回符合查询条件的记录数

返回参数说明

{"total":"9"}
添加文章

调用示例

{
    "action":"add_article",
    "title":"标题",
    "content":"内容",
    "typeid":"2"
}

参数说明

参数 是否必须 说明
action 是 操作方法 add_article
title 是 文章标题
content 是 文章内容
typeid 是 所属分类id

返回说明

返回写入文章的id

返回参数

{"article_id":"12"}

错误返回

{"errors":{"title":"标题必须","content":"内容必须","typeid":"分类必须"}}
删除文章

调用示例

{
    "action":"delete_article",
    "id":"25"
}

参数说明

参数 是否必须 说明
action 是 操作方法 delete_article
id 是 文章id

返回说明

返回code

返回参数

{"code":"101"}
获得一篇文章

调用示例

{
    "action":"get_article",
    "id":"25"
}

参数说明

参数 是否必须 说明
action 是 操作方法 get_article
id 是 文章id

返回说明

返回一条记录

返回参数

{"id":"6","title":"a","content":"a","addtime":"1431314306","typeid":"2","typename":"IOS","reid":"0"}
更新文章

调用示例

{
    "action":"update_article",
    "id":"6",
    "title":"标题",
    "content":"内容",
    "typeid":"2"
}

参数说明

参数 是否必须 说明
action 是 操作方法 update_article
id 是 文章id（条件）
title 是 标题
content 是 内容
typeid 是 所属分类id

返回说明

状态

返回参数

{"code":"101"}
返回码说明
code 说明
101 操作成功
102 操作失败
103 分类不存在
